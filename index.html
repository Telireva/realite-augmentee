Version corrigée — Autorise la caméra, puis vise le marqueur Hiro.

Mode professeur

1. Les étudiants ouvrent l'adresse HTTPS de cette page.
2. Ils autorisent la caméra.
3. Ils visent le marqueur Hiro.
4. La molécule H₂O apparaît en réalité augmentée.

QR code de l'activité

À scanner avec le cellulaire des étudiants.

Important : pour les cellulaires, l'adresse doit commencer par https://. Une adresse localhost ou http://192.168... n'est pas suffisante pour une classe.

Marqueur Hiro

À imprimer ou afficher pour faire apparaître la molécule.

Conseil : imprime le marqueur sur papier mat, garde-le bien éclairé et évite les reflets.
